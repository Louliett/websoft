S08

This assignment was performed with my project groupmate.

After analyzing three webpages: Imgur, GitHub and YouTube the results where charted on the following graphs.
Although we installed the Lighthouse extension, PageSpeed Insights is using Lighthouse lab data therefore we did not include Lighthouse average score. Judging the overall speed I was not surprised seing GitHub outrunning the other two pages. being mostly used by desktops, it just flies. Imgur, being a popular web platform for funny gifs and cats had quite a balanced speed between desktop and mobile version. I was surprised though seing YouTube, such a powerful and popular platform having the lowest PageSpeed score on both: mobile and desktop.

Opportunities for improvements for each page would be:
Imgur:
- Preload key requests (-2.91s);
GitHub:
- Serve images in next-gen formats (-0.91s);
- Deffer offscreen images (-0.8s);
YouTube:
- Preload key requests (-2.05s);
- Eliminate render-blocking resources (-1.4s)

I learned today about powerful optimization tools which will definitely come in handy when testing our own websites. While quite new in web development and not being introduced so much about performance, the fact that these tools are here to help us is surely worth looking into.
